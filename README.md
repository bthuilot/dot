# Dot files

My install script for both Arch Linux and MacOS

## macOS

*Last updated September 2021*


Install macOS via:
```shell
$ ./install_scripts/mac.sh
```

## Linux

*Last updated September 2021*


Install ArchLinux via:
```shell
$ ./install_scripts/arch.sh
```
